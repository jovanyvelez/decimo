¿Por qué las computadoras no hablan como nosotros?

Imagina que tienes que comunicarte con alguien que solo puede decir dos palabras: "sí" o "no". Con suficientes preguntas de sí/no puedes transmitir cualquier información del mundo. Eso es exactamente lo que hacen las computadoras.

Dato curioso: El chip del iPhone 15 tiene 19,000 millones de transistores en un espacio del tamaño de tu uña. Cada uno solo conoce el 0 y el 1.

El Sistema Decimal

Antes de aprender un nuevo idioma, hay que entender bien el propio. El sistema decimal funciona con potencias de 10.

¿Qué significa el número 3,472?

Concepto clave — Sistema Posicional: En cualquier sistema numérico, la posición de cada dígito le da su valor. Este principio aplica igual en binario, octal y cualquier otra base.
Verificación (3 × 1000) + (4 × 100) + (7 × 10) + (2 × 1)
= 3000 + 400 + 70 + 2
= 3,472 ✓

El Sistema Binario

El sistema binario funciona exactamente igual que el decimal, pero con potencias de 2 en lugar de potencias de 10.

Vocabulario esencial

Bit
Binary Digit

Un solo dígito binario (0 o 1). Es la unidad mínima de información digital.

Byte
8 bits

Grupo de 8 bits. Puede representar 256 valores distintos (0 a 255).

Las potencias de 2 que debes memorizar

Truco: cada valor es el doble del anterior. ¡Solo tienes que multiplicar por 2!

Conversión: Decimal → Binario

Regla de oro: Divide repetidamente entre 2 y anota los residuos de abajo hacia arriba.
Decimal → Binario Ejemplo 1 — Convertir 13 a binario
Verificación: 1×8 + 1×4 + 0×2 + 1×1 = 8 + 4 + 0 + 1 = 13 ✓
Decimal → Binario Ejemplo 2 — Convertir 25 a binario
25 ÷ 2 = 12 residuo → 1 12 ÷ 2 = 6 residuo → 0 6 ÷ 2 = 3 residuo → 0 3 ÷ 2 = 1 residuo → 1 1 ÷ 2 = 0 residuo → 1 ↑ leer de aquí

Resultado: 25₁₀ = 11001₂

1×16 + 1×8 + 0×4 + 0×2 + 1×1 = 16 + 8 + 0 + 0 + 1 = 25 ✓

Conversión: Binario → Decimal

Regla de oro: Multiplica cada dígito binario por su potencia de 2 según su posición (contando de derecha a izquierda desde 0), luego suma todo.
Binario → Decimal Ejemplo 3 — Convertir 10110₂ a decimal

Resultado: 10110₂ = 22₁₀ ✓

Binario → Decimal Ejemplo 4 — El byte máximo: 11111111₂
Posición: 7 6 5 4 3 2 1 0 Potencia: 128 64 32 16 8 4 2 1 Dígito: 1 1 1 1 1 1 1 1 128 + 64 + 32 + 16 + 8 + 4 + 2 + 1 = 255
Un byte tiene 8 bits y puede representar valores del 0 al 255. Por eso los colores RGB van de 0 a 255 en Photoshop o Canva.

El Sistema Octal

El sistema octal usa 8 como base y únicamente emplea los dígitos del 0 al 7. El dígito 8 y el 9 no existen en octal.

Uso real hoy: cuando ves chmod 755 en Linux, esos tres números son octales y controlan los permisos de un archivo.

Conversión: Decimal → Octal

¡Mismo método que binario! Divide entre 8 (no entre 2) y lee los residuos de abajo hacia arriba.
Decimal → OctalEjemplo 5 — Convertir 58 a octal
Verificación: 7×8¹ + 2×8⁰ = 56 + 2 = 58 ✓
Decimal → OctalEjemplo 6 — Convertir 200 a octal
200 ÷ 8 = 25 residuo → 0 25 ÷ 8 = 3 residuo → 1 3 ÷ 8 = 0 residuo → 3 ↑ leer de aquí → 310₈

Verificación: 3×64 + 1×8 + 0×1 = 192 + 8 + 0 = 200 ✓

Conversión: Octal → Decimal

Octal → DecimalEjemplo 7 — Convertir 135₈ a decimal

Resultado: 135₈ = 93₁₀ ✓

Binario ↔ Octal — El Truco de los Grupos de 3

El secreto: Cada dígito octal equivale exactamente a 3 bits (porque 2³ = 8). ¡Sin divisiones, sin multiplicaciones, solo agrupar!
Truco grupos 3Ejemplo 8 — Convertir 110101₂ a octal

La Computadora Humana

Contexto del juego: El año es 1960. Los transistores no llegan a tiempo para construir el procesador. Ustedes son el procesador. La misión: convertir números sin errores o el cohete espacial se pierde. 🚀

Los 5 Roles

ROL 1
Generador de Entrada
Escribe en una tarjeta el número decimal que la "computadora" debe procesar. Solo puede usar números del 0 al 127.
"Número ingresado al sistema."
ROL 2
El Divisor
Aplica el método de divisiones sucesivas entre 2. Entrega una hoja con todas las divisiones y residuos anotados.
"Procesando… división completada."
ROL 3
Lector de Bits
Lee los residuos de abajo hacia arriba y escribe el número binario en una tarjeta grande para el grupo.
"Bits leídos. Transmitiendo señal binaria."
ROL 4
Agrupador Octal
Toma el número binario, agrupa en bloques de 3 desde la derecha y convierte cada grupo a su dígito octal.
"Comprimiendo a octal. Compresión exitosa."
ROL 5
El Verificador
Comprueba con la Tabla Maestra que la cadena sea correcta. Si detecta error, activa la alarma: "¡ERROR DE SISTEMA!" y el proceso se reinicia.
"Verificación completa. Sistema operativo." ✓

Las 4 Rondas

Ronda 1 · 5 min
Entrenamiento guiado por el docente con el número 42. Cada rol practica en voz alta.
Ronda 2 · 8 min
Primera misión real. El Generador crea un número y cada rol ejecuta su función. Victoria: cero errores.
Ronda 3 · 8 min
Misión inversa: el Generador escribe un número octal. El proceso se invierte: Oct → Bin → Dec.
Ronda 4 · 7 min
Torneo: dos grupos compiten con el mismo número. Gana quien llegue al resultado correcto primero.

Resumen y Cierre

Las 5 Reglas de Oro

1
Regla del 0 y el 1
Todo en una computadora es binario. El resto de sistemas son solo formas más cómodas de leerlo.
2
Regla de las Divisiones
Para convertir decimal a cualquier base, divide entre esa base y lee los residuos de abajo hacia arriba.
3
Regla de las Potencias
Para convertir a decimal, multiplica cada dígito por su potencia de posición (contando de derecha a izquierda desde 0) y suma.
4
Regla de los Grupos de 3
Para convertir entre binario y octal, agrupa de 3 en 3 desde la derecha. Un dígito octal = 3 bits exactos.
5
Regla de la Verificación
Siempre verifica tu respuesta convirtiendo de regreso al número original. Si no coincide, hay un error en el proceso.

Glosario

Bit
Dígito binario. Unidad mínima de información digital (0 o 1).
Byte
Grupo de 8 bits. Puede representar 256 valores (0 a 255).
Nibble
Grupo de 4 bits. La mitad de un byte.
Base
Cantidad de símbolos distintos que usa un sistema numérico.
Sistema posicional
Sistema donde el valor de un dígito depende de su posición en el número.
Transistor
Componente electrónico que actúa como interruptor. Base del hardware digital.
Residuo
Lo que sobra después de una división entera.
Binario
Sistema de base 2. Solo usa 0 y 1. El lenguaje nativo de las computadoras.
Octal
Sistema de base 8. Usa dígitos del 0 al 7. Muy relacionado con el binario.
